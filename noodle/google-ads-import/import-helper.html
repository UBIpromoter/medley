Before you start: In Google Ads Editor, delete the two partially-posted campaigns (Teachers & Admins). Right-click each → Remove. Then Post that removal. Start fresh.
1 Campaigns 2 rows
Navigate to: Campaigns in left panel (under "All campaigns")
  1. Click Make multiple changes (top toolbar, or Edit menu)
  2. Select "Use selected destinations" or "My data includes columns for campaigns and/or ad groups"
  3. Click Copy below, then Paste from clipboard in the dialog
  4. Click Process → review → Finish and review changes
2 Ad Groups 2 rows
Navigate to: Ad groups in left panel (under the campaign tree)
  1. Click Make multiple changes
  2. Check "My data includes columns for campaigns and/or ad groups"
  3. Click Copy below, then Paste from clipboard
  4. Click Process → review → Finish and review changes
3 Keywords 39 rows
Navigate to: Keywords and targeting → Keywords in left panel
  1. Click Make multiple changes
  2. Check "My data includes columns for campaigns and/or ad groups"
  3. Click Copy below, then Paste from clipboard
  4. Click Process → review → Finish and review changes
4 Responsive Search Ads 2 rows
Navigate to: Ads → Responsive search ads in left panel
  1. Click Make multiple changes
  2. Check "My data includes columns for campaigns and/or ad groups"
  3. Click Copy below, then Paste from clipboard
  4. Click Process → review → Finish and review changes
5 Negative Keywords 109 rows
Navigate to: Keywords and targeting → Keywords, Negative in left panel
  1. Click Make multiple changes
  2. Check "My data includes columns for campaigns and/or ad groups"
  3. Click Copy below, then Paste from clipboard
  4. Click Process → review → Finish and review changes
After all 5 are pasted: Click Post (top toolbar) → All campaigns → Post. Everything goes up in one shot. Campaigns start Paused — enable them when you're ready to spend.
